# Iris-PAD
## Environment Settings
Please provide the csv path and dataset path according to your environment. The ground truth of the dataset is train,imageName,class.
An example is demonstrated for the clarkson dataset.
## Grad-Cam Images
In order to use gradcam please use 
```bash
pip install grad-cam
```
## Generate pyeer reports.
In order to use pyeer please use 
```bash
pip install pyeer
```
